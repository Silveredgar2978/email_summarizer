# Email Summarizer using Gemini AI (in python)

This is my first email summarizer AI tool.

## What it does

- reads mail
- organizes content
- shows helpful summary for any mail

## Requirements 

- Python 3.10+
- A Gemini API key

## How to run 

1. Clone this repository
2. Install dependencies:
   pip install -r requirements.txt
3. Add your Gemini API key in a .env (just like in .env.example)
4. Run the program: summarizer.py
